### Critical: `@/*` alias

`tsconfig.json` maps `@/*` to the **project root** (`./`), not `./src/`:

```ts
"paths": { "@/*": ["./*"] }
```

This means:
- `@/components/ui/button` → `components/ui/button.tsx` (root)
- `@/lib/utils` → `lib/utils.ts` (root)
- `@/src/app/...` → `src/app/...` (if you ever need it)

**Never** place shared code in `src/lib/` expecting `@/lib/` to resolve there — it won't.

## Database Schema (prisma/schema.prisma)

Prisma client output goes to `generated/prisma/` (non-standard — import from there, not `@prisma/client`).

### Models

| Model | Purpose |
|---|---|
| `User` | BetterAuth user — email/OAuth identity |
| `Account` | OAuth provider accounts linked to a User |
| `Session` | BetterAuth session tokens |
| `Verification` | Email/OTP verification codes |
| `Portfolio` | One user can own multiple NEPSE portfolios (own, spouse, parent…) |
| `Transaction` | Each buy or sell event: quantity, price, charges, net amount |

### TransactionType enum: `BUY | SELL`

### Charge fields on Transaction (stored at save time so historical records survive rate changes)
- `brokerCommission` — tiered by transaction value (see env vars)
- `dpCharge` — flat NPR 25 per transaction
- `sebon` — 0.015% of transaction value
- `capitalGainTax` — 7.5% (≤ 365 days held) or 5% (> 365 days); 0 for BUY
- `netAmount` — derived total (stored for fast queries)

## Environment Variables

### `.env` (real secrets, never commit)

| Key | Purpose |
|---|---|
| `DATABASE_URL` | Neon PostgreSQL connection string |
| `BETTER_AUTH_SECRET` | BetterAuth signing secret |
| `BETTER_AUTH_URL` | App base URL for BetterAuth (e.g. `http://localhost:3000`) |

### `.env.local` (public constants + OAuth placeholders)

| Key | Value | Purpose |
|---|---|---|
| `DATABASE_URL` | _(fill)_ | Override for local dev if different from .env |
| `BETTER_AUTH_SECRET` | _(fill)_ | Override placeholder |
| `GOOGLE_CLIENT_ID` | _(fill)_ | Google OAuth client ID |
| `GOOGLE_CLIENT_SECRET` | _(fill)_ | Google OAuth client secret |
| `NEXT_PUBLIC_APP_URL` | `http://localhost:3000` | Public base URL |
| `NEXT_PUBLIC_CGT_SHORT_TERM` | `0.075` | 7.5% CGT for shares held ≤ 365 days |
| `NEXT_PUBLIC_CGT_LONG_TERM` | `0.05` | 5% CGT for shares held > 365 days |
| `NEXT_PUBLIC_DP_CHARGE` | `25` | NPR 25 flat DP charge per transaction |
| `NEXT_PUBLIC_SEBON_RATE` | `0.00015` | 0.015% SEBON fee on transaction value |
| `NEXT_PUBLIC_BROKER_RATE_UPTO_50K` | `0.004` | 0.40% broker commission up to NPR 50k |
| `NEXT_PUBLIC_BROKER_RATE_50K_500K` | `0.0037` | 0.37% broker commission NPR 50k–500k |
| `NEXT_PUBLIC_BROKER_RATE_ABOVE_500K` | `0.0034` | 0.34% broker commission above NPR 500k |

`NEXT_PUBLIC_*` vars are safe to expose to the browser — they're regulatory constants, not secrets. Edit them here when NEPSE changes the fee schedule.

## Conventions Established

- **`toast` is deprecated** in shadcn v4 — use `sonner` (`components/ui/sonner.tsx`) and the `toast()` function from `sonner` package.
- **shadcn style** is `base-nova` (Tailwind v4 default picked by `shadcn init -d`). Components use `@base-ui/react` primitives, not `@radix-ui/react`.
- **`form.tsx` was written manually** — `bunx shadcn@latest add form` hangs in this environment. The file at `components/ui/form.tsx` follows the same base-nova patterns and wraps `react-hook-form`'s `FormProvider`/`Controller`.
- **No migrations yet** — schema is defined but `prisma migrate dev` has not been run. Run it once DATABASE_URL is confirmed and the schema is finalised.
- **Fonts**: Geist Sans (`--font-geist-sans`) and Geist Mono (`--font-geist-mono`) loaded via `next/font/google` in `src/app/layout.tsx`.
